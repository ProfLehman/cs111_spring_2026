## History Dates 

- 1800’s Charles Babbage "idea" for mechanical general purpose computing device “analytical engine”
- 1800’s Ada Lovelace considered “first programmer”
- 1940’s First Digital Computers (ENIAC), Zuse (German mechanical computer)
- 1950’s First Commercial Computers
- 1970’s First Personal Computers
- Describe Hardware Progression from earliest to latest (Vacuum Tube, Transistor, Integrated Circuit)

## Binary, Decimal, Hex, ASCII/Unicode
- Convert binary (base 2), decimal (base 10), and hex (base 16) numbers 
- Define ASCII (extended ASCII), Unicode
- Decode short ASCII message (given ASCII code for “A” = 65 = 0100 0001)

## Data Representation
- Define analog vs digital information
- Define: bit (1 or 0), byte (8 bits)
- Define: kilobyte (1,000 bytes), megabyte (1,000,000 bytes), gigabyte (1,000,000,000), terabyte (1,000,000,000,000), petabyte (1,000,000,000,000,000)
- Define Lossy Compression (JPG, MP3) vs. Lossless Compression (ZIP, TIFF, BMP, WAV), compress,  uncompressed, ZIP file concept
- Using predefined Huffman code, show coding of word using Huffman compression
- Recognize common image formats (JPG, PNG, GIF, HEIC)
- Describe a common file format and extension (e.g. Word .docx, Excel .xlsx, .PDF, .mp3, .txt)

## Logic
- Concept: Switch electricity “On” represents 1 and “Off” represents 0
- Create and recognize truth tables for basic logic gates (NOT, AND, OR, XOR) Note: will not include NAND, NOR
- Create complete truth tables for a given circuit (i.e. show outputs for all inputs)

## Hardware
- Define: CPU, CPU core, graphics processor, RAM, clock frequency/speed (Gigahertz Ghz)
- Relative speeds of registers, cache, memory (RAM), and hard drive (SSD and mechanical)
- Define types of computers Supercomputer, Server, Desktop, Laptop, Tablet, Smartphone, Embedded
- Define Moore’s Law (technology doubles every 1.5 to 2 years)
- Describe Current Desktop Standards: Memory (8 GB - 64 GB), Storage (256 GB – 2 TB), CPU (2 Ghz - 5 Ghz)
